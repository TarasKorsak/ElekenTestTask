# This is my solution for Eleken test task

## Build Setup

``` 

git clone https://github.com/TarasKorsak/ElekenTestTask.git

cd ElekenTestTask

# install dependencies
npm install

# serve at localhost:3000
gulp dev

# build for production with minification
gulp prod
```
